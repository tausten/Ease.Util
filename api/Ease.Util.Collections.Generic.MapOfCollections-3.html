Class MapOfCollections<TKey, TCollection, TValue>

A mapping of keys to a collection of values. Takes care of ensuring that an empty collection is present, avoiding need for null checks and collection initialization per key prior to use. Eg:

var families = new MapOfCollections<string, HashSet<string>, string>();
families["Doe"].Add("Jane");
families["Doe"].Add("John");
var theDoes = string.Join(", ", families["Doe"]);

See MapOfHashSets<TKey, TValue> and MapOfLists<TKey, TValue> for convenient simplification for common some cases.
