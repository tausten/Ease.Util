Class FactoryScoped<T>

A utility class for managing lifetime of factory pattern managed resources that require to be released in some way other than by the simple Dispose pattern. The allocation is done lazily such that if the .Instance member is not accessed, then neither allocateFunction nor releaseAction will be called.
