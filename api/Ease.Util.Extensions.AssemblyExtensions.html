GetResourceBySuffixAsStream(Assembly, String)

Returns a Stream of the embedded resource matching the passed resourceNameSuffix. If there is no case-sensitive match, then case-insensitive match (if any) will be used. If there are multiple matches, the resource with the shortest ManifestResourceName will be used. If there are no matches, null is returned.

GetResourceBySuffixAsString(Assembly, String, Encoding)

Returns the contents of the embedded resource matching the passed resourceNameSuffix as a string (read as the specified encoding). If there is no case-sensitive match, then case-insensitive match (if any) will be used. If there are multiple matches, the resource with the shortest ManifestResourceName will be used. If there are no matches, null is returned.
