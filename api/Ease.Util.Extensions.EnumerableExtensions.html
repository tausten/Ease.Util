ImmediateBatch<TElement>(IEnumerable<TElement>, Int32, Action<IEnumerable<TElement>>)

Break the elements up into batches of size batchSize, optionally executing the passed batchOperation on each batch before returning it. All batches are assembled, and batchOperation executions performed immediately.

YieldedBatch<TElement>(IEnumerable<TElement>, Int32, Action<IEnumerable<TElement>>)

Break the elements up into batches of size batchSize, optionally executing the passed batchOperation on each batch before yield returning it.

NOTE: If the caller doesn't iterate over the returned IEnumerable of batches, no batchOperation will be executed because this is lazily iterated. If you want the operations to be executed immediately on all batches, then use ImmediateBatch<TElement>(IEnumerable<TElement>, Int32, Action<IEnumerable<TElement>>) or be sure to iterate over the returned IEnumerable immediately.
