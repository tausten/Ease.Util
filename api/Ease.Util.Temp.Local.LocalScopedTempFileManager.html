LocalScopedTempFileManager(DirectoryInfo)

Temporary files will be managed under the specified directoryInfo. The manager assumes it takes ownership of the directory (and any subdirectories) and will delete them when Dispose(..) is called, so do not attempt to have multiple managers operating on the same folder or non-deterministic behavior will occur.
